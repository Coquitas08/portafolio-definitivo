
Este es mi portafolio web desarrollado con **HTML, CSS y JavaScript**. El objetivo de este sitio es poder mostrar mi perfil de forma ordenada (fin)

## Secciones principales

- **Inicio / Sobre mí**: Presentación breve con información personal y de contacto.  
- **Proyectos**: Muestra algunos de los trabajos y prácticas realizadas (en el colegio)
- **Habilidades**: Idiomas, conocimientos en programación, entre otras cosas. 
- **Películas favoritas**: Un espacio más personal para mostrar gustos e intereses.

## Tecnologías utilizadas

- **HTML5**  
- **CSS** (diseño responsive - el que mejora la experiencia del usuario - )  
- **JavaScript** (efectos básicos e interacción en las secciones de proyectos y demás)  

## Modo de uso

1. Descargar o clonar el repositorio.  
2. Abrir el archivo `index.html` en cualquier navegador.
